Naming alkyl halides

All the same rules apply when naming the alkyl halides as for naming the hydrocarbons. We will only be dealing with the haloalkanes (i.e. there are no other functional groups). The halide atom is treated in the same way as a branched group.

Naming the haloalkanes

Give the IUPAC name for the following compound:

<code>
		    (-4.2,-1.2)(0.2,1.3)
		    \rput(-4,0){\textbf{H}}
		    \psline(-3.8,0)(-3.2,0)
		    \rput(-3,0){\textbf{C}}
		    \psline(-3,0.2)(-3,0.8)
		    \rput(-3,1){\textbf{H}}
		    \psline(-3,-0.2)(-3,-0.8)
		    \rput(-3,-1){\textbf{H}}
		    \psline(-2.8,0)(-2.2,0)
		    \rput(-2,0){\textbf{C}}
		    \psline(-2,0.2)(-2,0.8)
		    \rput(-2,1){\textbf{\red{\large Cl}}}
		    \psline(-2,-0.2)(-2,-0.8)
		    \rput(-2,-1){\textbf{H}}
		    \psline(-1.8,0)(-1.2,0)
		    \rput(-1,0){\textbf{C}}
		    \psline(-1,0.2)(-1,0.8)
		    \rput(-1,1){\textbf{H}}
		    \psline(-1,-0.2)(-1,-0.8)
		    \rput(-1,-1){\textbf{H}}
		    \psline(-0.8,0)(-0.2,0)
		    \rput(0,0){\textbf{H}}
		  </code>
Identify the functional group

There is a halogen atom and no other functional group. This compound is therefore a haloalkane, and will have the suffix -ane.

Find the longest carbon chain containing the functional group

There are three carbons in the longest chain containing the halogen atom. The prefix is prop-.

Number the carbons in the longest chain

You need to number the carbons so that the halide atom is on the carbon atom with the lowest number. In this case you can number from either side.

Name the halide atom and assign the number for the carbon atom it is attached to

The halogen is a chlorine atom. It is attached to carbon number 2 and so will have the name 2-chloro.

Look for branched groups

There are no branched groups in this compound.

Combine the elements of the name into a single word in the following order: halide atoms; prefix; name ending according to functional group

The name of the compound is 2-chloropropane.

Number the carbons in the longest chain

You need to number the carbons so that the halide atoms are on the carbon atoms with the lowest numbers. You must number from left to right here so that one halogen atom is on carbon 1 and two halogen atoms are on carbon 3.

The name of the compound is 1,3-dibromo-3-fluorobutane. Note that we place the halogens in alphabetical order: bromo (ignore the di/tri/tetra) is before fluoro.
